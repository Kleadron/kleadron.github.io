OpacityMonkey

OpacityMonkey is a command-line tool to set the alpha of all pixels in an image to 1.
Preferably works with PNG files.

Instructions

In command prompt, type "OpacityMonkey.exe infile.png outfile.png".
Infile and outfile can be named whatever you want, but outfile will be created by OpacityMonkey.

Preferably, use PNG files. The output file is always in PNG format.
