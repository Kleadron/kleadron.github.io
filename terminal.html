The Command Line Challenge

This is intended for people who think Linux is the best thing ever.

Completely ignore or remove the terminal on a computer running the Linux distro Ubuntu, and the same for the command prompt and powershell from a Windows PC.

Stub your toes for every single time you require the terminal on Ubuntu, to do something you can do on Windows via the GUI.

Assess the damages and decide if Linux distros really are user friendly.

If you do not have any toes left, or possibly have developed a habit of eating mysterious matter from your foot, consider making Linux distributions a much more user friendly and graphical experience.
