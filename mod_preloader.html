Texture Pre-Loader for Minecraft 1.2.5

Finds textures being loaded at runtime and instead loads them at startup on the loading screen the next time you start Minecraft. Good for Hi-res texture packs so no babies are thrown into the gears when you open an inventory and some craplord forgot to pre-load their GUI textures.

Effects are not immediate. Open interfaces and stuff to allow the pre-loader to find the textures. Check config file to configure log output.

Features

  • Longer initial load time for smoother gameplay
  • Fixes initial lag of opening new interfaces or scrolling through the item list

Downloads

I expect you to know how to install mods. Just drag the zip file into your mods folder.
