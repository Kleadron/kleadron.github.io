About the website

I thought it would be fun to make a website in old style. So here it is =]

This website is designed with the HTML 4.01 specification in mind, so it should at least run on Internet Explorer 5, maybe 4. If your browser doesn't support it, or javascript, HA. And HOW.

This is made for a generic resolution of 1024x768.
If the website doesn't look right, get a bigger monitor or increase your resolution.

Some parts of the website now use HTML5 components for convenience, but would still work on crusty old browsers. Since this is hosted on Github, HTTPS is enforced.
