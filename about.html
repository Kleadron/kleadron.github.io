About the website

I thought it would be fun to make a website in old style. So here it is =]

This website was designed with the HTML 4.01 specification in mind, so it should at least run on Internet Explorer 5, maybe 4. I've recently tested it with IE11 and Firefox 2.

Some parts of the website now use HTML5 components for convenience, but should show a fallback on crusty old browsers.

It will break somewhat if javascript is disabled, since I use that to insert the header and footer.
